# Clinica App

## Descripción
**Clinica App** es un sistema integral de gestión para clínicas, que permite administrar:  

- **Especialidades médicas**  
- **Distritos**  
- **Médicos**  
- **Pacientes**  
- **Citas médicas**  

El proyecto está compuesto por un **backend en Node.js** con Express y MySQL (Sequelize) y un **frontend en React con Vite**, utilizando **Bootstrap** y **FontAwesome** para un diseño moderno, responsivo y fácil de usar.

---

## Objetivo
Proporcionar una plataforma completa para la administración clínica, permitiendo:  

- Registrar, actualizar y eliminar especialidades médicas.  
- Gestionar distritos.  
- Administrar información de médicos y pacientes.  
- Crear y controlar citas médicas, incluyendo pagos y estado de atención.  
- Contar con un dashboard administrativo con sidebar, navegación intuitiva y tablas dinámicas.
